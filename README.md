# Weekly Temperatures Program

This Java program demonstrates the use of ArrayLists, loops, decision structures, and methods. The program stores the days of the week and their corresponding average temperatures in two separate ArrayLists.

The user can enter a specific day of the week to view that day's temperature or enter **week** to display all temperatures and the weekly average. Input validation is included to ensure only valid entries are accepted.

## Concepts Demonstrated

* ArrayLists
* Loops (`while` and `for`)
* Decision structures (`if-else`)
* Methods and modular design
* User input with the Scanner class
* Input validation
